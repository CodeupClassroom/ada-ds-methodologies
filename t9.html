defaultdict lets us create a dictionary that has default values. That is, if we try to access a key that doesn’t exist in a defaultdict, the default value will be returned.

from collections import defaultdict
#

Setup our beginning data structure that represents phone keys

keypad = {
    1: '',
    2: 'abc',
    3: 'def',
    4: 'ghi',
    5: 'jkl',
    6: 'mno',
    7: 'pqrs',
    8: 'tuv',
    9: 'wxyz',
    0: ' ',
}
#

Now we will transform keypad so that we create a new dictionary, keypresses that we can use to find out the number of keypresses required for a given letter. The keys in keypresses will be the number of times a number must be pressed, and the values will be a list of the characters that require that many keypresses.

We’ll make keypresses a default dictionary with a default value of an empty list, so that we can add characters to each list.

keypresses = defaultdict(lambda: [])
for key, letters in keypad.items():
    for i, l in enumerate(letters):
        n_presses = i + 1
        keypresses[n_presses] += l
#

Now that we have our data structure setup, we’ll implement the keypress
counting with two separate functions, one that gets the number of keypresses
for a single letter, and one that uses this function to get the total number
of keypresses for a word.

    

    

      
def get_n_keypresses_for_letter(l):
    for n_presses, chars in keypresses.items():
        if l in chars:
            return n_presses

    return sum([get_n_keypresses_for_letter(letter) for letter in word])
